## 用法

1. 使用 npx 安装

```
npx @juyren/library-cli <projectName> <--template (javascript | typescript)>
```

2. 使用全局安装

```
npm install @juyren/library-cli -g

library-cli <projectName> <--template (javascript | typescript)>
```

## 选项

| template   |              |
| ---------- | ------------ |
| javascript | 基于 js 开发 |
| typescript | 基于 ts 开发 |
